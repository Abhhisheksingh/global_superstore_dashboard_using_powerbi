# global_superstore_dashboard_using_powerbi

Overview

This repository contains the Global Superstore dataset along with a set of business questions designed to build an interactive Power BI dashboard. The project focuses on analyzing global sales performance, customer behavior, and profitability across different regions, categories, and time periods.
